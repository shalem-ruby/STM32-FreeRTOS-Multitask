# STM32 FreeRTOS Multitasking System

## Description
FreeRTOS based multitasking system on STM32F103C8T6.
Two tasks running simultaneously managed by FreeRTOS scheduler.

## Tasks
- Task 1: LED toggles every 500ms on PA5
- Task 2: UART sends "FreeRTOS Running..." every 1 second

## Tools Used
- STM32CubeIDE
- FreeRTOS CMSIS V1
- Embedded C
- HAL Library

## Skills Demonstrated
- RTOS Task Management
- UART Communication
- GPIO Control
- STM32 HAL Library
- Embedded C Programming

## Board
STM32F103C8T6
